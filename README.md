# Healthcare Provider (HCP) Claim Volume Analysis
**Course:** BIA 810 - Big Data Technologies @ Stevens Institute of Technology

## Project Overview
This project analyzes the claim volume among Healthcare Providers from 2016-2018 to optimize Sales Force ROI.

## Key Insights
* **Long-tail Distribution:** Discovered a massive disparity between HCP segments and their actual claim volume.
* **Strategic Recommendation:** Transitioned low-volume HCPs to digital-only engagement (NPP) to preserve high-touch resources for top-tier targets.

## Technology Used
* Python (Pandas, Matplotlib/Seaborn for visualization)
* Jupyter Notebook (Exported as HTML)
